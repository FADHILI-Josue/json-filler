## Privacy Policy for Swagger Fake Filler

**Last updated:** 15th November 2025

Json Filler ("the Extension") is a browser extension created by FADHILI Josue.

**Data Collection**
The Extension does not collect, store, share, or transmit any user personal information, browsing history, or form data to external servers. All data processing (parsing JSON and generating dummy data) happens locally on your device within the browser.

**Permissions**
The Extension uses permissions (activeTab, scripting, contextMenus) solely to perform its function of filling input fields when triggered by the user.

**Contact**
For questions, please create an issue on the GitHub repository.
